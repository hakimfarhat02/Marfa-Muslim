🌟 مقدمة عن القالب

قالب "الوقار الرقمي" هو قالب بلوجر احترافي مصمم خصيصاً للمواقع الإسلامية والعلمية. يتميز القالب بتصميم عصري وأنيق يجمع بين الأصالة الإسلامية والحداثة التقنية، مع دعم كامل للغة العربية والاتجاه من اليمين إلى اليسار (RTL).

🎯 ما الجديد في النسخة 2.0؟

  • دعم كامل لـ مصمم المظاهر (Theme Designer) لتغيير الألوان والخطوط بدون تعديل الكود
  • ربط أداة الطقس بـ OpenWeatherMap API للحصول على بيانات حية
  • إضافة خيار المدينة اليدوية لمواقيت الصلاة
  • جعل البسملة التلقائية اختيارية
  • تحسينات SEO مع Breadcrumbs Schema
  • تحميل غير متزامن (Async/Defer) للسكربتات لتحسين السرعة
  • حفظ حالة الوضع المظلم في localStorage

⚙️ خطوات التثبيت

الدخول إلى لوحة تحكم بلوجر

افتح موقع blogger.com وسجل الدخول بحسابك.

اختيار المدونة

اختر المدونة التي تريد تطبيق القالب عليها.

الذهاب إلى قسم المظهر

من القائمة الجانبية، اضغط على "المظهر" أو "Theme".

رفع القالب

اضغط على السهم المنسدل بجوار "تخصيص" ثم اختر "تحرير HTML". احذف كل الكود الموجود والصق كود القالب الجديد، ثم احفظ التغييرات.

التحقق من التثبيت

افتح المدونة في نافذة جديدة للتأكد من أن القالب يعمل بشكل صحيح.

🎨 استخدام مصمم المظاهر (Theme Designer)

يدعم القالب مصمم المظاهر المدمج في بلوجر، مما يتيح لك تخصيص مظهر الموقع بدون الحاجة لتعديل الكود. للوصول إليه:

لوحة التحكم ← المظهر ← تخصيص

المتغيرات المتاحة:

المتغير الوصف القيمة الافتراضية
اللون الأساسي لون الهيدر والقائمة والعناصر الرئيسية #1b3a2d (أخضر داكن)
اللون الذهبي لون التمييز والأيقونات #c9a84c (ذهبي)
خلفية الموقع لون خلفية الصفحة #fdfdfd (أبيض مائل للكريمي)
لون النصوص لون النصوص الرئيسية #222222 (رمادي داكن)
خط العناوين الخط المستخدم في العناوين Amiri
خط المحتوى الخط المستخدم في نصوص المقالات Amiri
خط الواجهة الخط المستخدم في عناصر الواجهة Tajawal
إظهار البسملة إظهار/إخفاء البسملة التلقائية مفعّل

🌤️ إعدادات الطقس (OpenWeatherMap)

أداة الطقس في القالب تستخدم OpenWeatherMap API لجلب بيانات الطقس الحية. للحصول على مفتاح API مجاني:

إنشاء حساب في OpenWeatherMap

اذهب إلى openweathermap.org/api وسجل حساباً جديداً.

الحصول على مفتاح API

بعد تسجيل الدخول، اذهب إلى قسم "API Keys" وانسخ المفتاح الخاص بك.

إضافة المفتاح للقالب

في مصمم المظاهر، أدخل المفتاح في حقل "مفتاح OpenWeatherMap API".

الخطة المجانية تتيح لك 1000 طلب يومياً، وهو أكثر من كافٍ لموقع بلوجر.
لا تشارك مفتاح API الخاص بك مع الآخرين لتجنب استهلاك الحصة اليومية.

🕌 إعدادات مواقيت الصلاة

القالب يستخدم Aladhan API لجلب مواقيت الصلاة تلقائياً بناءً على موقع الزائر. لكن يمكنك تعيين مدينة افتراضية يدوياً:

الإعدادات المتاحة:

الإعداد الوصف مثال
المدينة الافتراضية اسم المدينة الافتراضية مكة المكرمة
خط العرض الافتراضي Latitude للمدينة 21.3891
خط الطول الافتراضي Longitude للمدينة 39.8579
يمكنك الحصول على إحداثيات أي مدينة من Google Maps بالنقر بزر الماوس الأيمن على الموقع.

✨ إعدادات البسملة التلقائية

البسملة التلقائية هي ميزة فريدة في القالب تضيف "بسم الله الرحمن الرحيم" تلقائياً في بداية كل مقال. يمكنك التحكم فيها من مصمم المظاهر:

  • لإظهار البسملة: اترك الخيار "إظهار البسملة التلقائية" مفعّلاً
  • لإخفاء البسملة: عطّل الخيار "إظهار البسملة التلقائية"
حتى لو أخفيت البسملة التلقائية، يمكنك إضافتها يدوياً في نص المقال.

⭐ المميزات الرئيسية

📱 تصميم متجاوب (Responsive)

القالب يتكيف تلقائياً مع جميع أحجام الشاشات: الهواتف، الأجهزة اللوحية، وأجهزة الكمبيوتر.

🌙 الوضع المظلم (Dark Mode)

زر تبديل في الشريط العلوي يسمح للزائر بالتبديل بين الوضع الفاتح والمظلم. يتم حفظ التفضيل في المتصفح.

🔍 تحسين محركات البحث (SEO)

بيانات Schema.org المنظمة، Open Graph، Twitter Cards، وعناوين ديناميكية لكل صفحة.

⚡ سرعة التحميل

تحميل غير متزامن للسكربتات، صور ذكية مع lazy loading، وخطوط محسّنة.

🎵 مشغل الصوتيات المدمج

يكتشف روابط الملفات الصوتية في المقال تلقائياً ويعرض مشغلاً أنيقاً.

📖 أدوات القراءة

أزرار تكبير/تصغير الخط، زر الطباعة، وشريط تقدم القراءة.

🔍 تحسين محركات البحث (SEO)

القالب يتضمن عدة ميزات SEO متقدمة:

البيانات المنظمة (Schema Markup):

  • Article Schema: لصفحات المقالات مع تفاصيل الكاتب والناشر
  • BreadcrumbList Schema: لمسار التنقل في الصفحات
  • WebSite Schema: مع مربع بحث مدمج

وسوم Meta:

  • Open Graph Tags لمشاركة أفضل على فيسبوك
  • Twitter Cards لمشاركة أفضل على تويتر
  • عناوين ديناميكية حسب نوع الصفحة
  • أوصاف مخصصة لكل صفحة

الروابط:

  • روابط Canonical لمنع المحتوى المكرر
  • روابط دائمة (Permalinks) محسّنة

⚡ تحسين الأداء

تم تنفيذ عدة تقنيات لتحسين سرعة التحميل:

التقنية الوصف
defer تأجيل تحميل JavaScript حتى اكتمال تحميل HTML
async تحميل السكربتات بشكل غير متزامن
preconnect إنشاء اتصال مسبق مع خوادم الموارد
display=swap عرض النص فوراً قبل تحميل الخطوط
loading="lazy" تحميل الصور عند الاقتراب منها

🔧 التخصيص المتقدم

تغيير الألوان عبر CSS:

إذا كنت ترغب في تخصيص ألوان إضافية، يمكنك إضافة CSS مخصص:

/* المظهر ← تخصيص ← إضافة CSS مخصص */ :root { --primary-color: #1b3a2d; --accent-color: #c9a84c; }

تعديل شريط التذكير:

يمكنك تعديل الآيات والأحاديث في شريط التذكير من أداة HTML6 في تخطيط القالب.

إضافة أقسام جديدة:

يمكنك إضافة أقسام جديدة من خلال لوحة التحكم ← المظهر ← تخطيط.
